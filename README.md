# E-Commerce Sales Analysis

## Overview
This project analyzes an online retail dataset to understand customer behavior, sales trends, and product performance.

## Dataset
The dataset contains 541,000+ e-commerce transactions including product information, quantities, prices, and customer IDs.

## Tools Used
- Python (pandas, numpy)
- PostgreSQL
- Tableau
- Scikit-learn

## Analysis
The project includes:

• Data cleaning and preprocessing  
• Exploratory data analysis  
• RFM customer segmentation  
• SQL queries for revenue insights  
• Tableau dashboard visualization  
• Logistic regression model predicting repeat purchases  

## Key Insights

- A small group of customers generates a large portion of revenue
- Sales peak during certain months indicating seasonal demand
- A small number of products dominate total revenue
- Higher spending customers are more likely to become repeat buyers

## Dashboard
Tableau dashboard visualizing sales trends, top products, and geographic sales.
